## Funcionalidades Implementadas

### Entidades
- **Cliente**: Cadastro com nome e data de clienteDesde
- **Produto**: Cadastro com nome, preço e controle de estoque
- **Pedido**: Relaciona cliente + produto + quantidade

### Endpoints CRUD (Obrigatórios)

- CLIENTES: POST/PUT/GET/GET{id}/DELETE /clientes(/{id})
- PRODUTOS: POST/PUT/GET/GET{id}/DELETE /produtos(/{id})
- PEDIDOS: POST/PUT/GET/GET{id}/DELETE /pedidos(/{id})
